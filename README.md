# muscat-web-leads

Live demo websites for Muscat businesses (Abdulkhalid's web-agency venture).
Each business has its own folder at the **repo root** so GitHub Pages serves clean links:

- `nabu-cafe/`             → https://khlied202.github.io/muscat-web-leads/nabu-cafe/
- `sweetylicious-bakery/`  → https://khlied202.github.io/muscat-web-leads/sweetylicious-bakery/
- `fellas-and-co-barbershop/` → https://khlied202.github.io/muscat-web-leads/fellas-and-co-barbershop/

Each folder contains `index.html` (the site), `assets/` (images/CSS), and `message.txt`
(the WhatsApp pitch used for outreach). The root `index.html` is a simple landing page
linking to all demos. `.nojekyll` disables Jekyll so files are served as-is.

**Source of truth:** the Desktop working copy `Muscat-Web-Agency/businesses/<slug>/`.
This repo is a flattened mirror of that folder. Keep them in sync.

Pages: main branch, root (`/`).
